# Module: button_interrupt_controlled_led

## Purpose

Control an LED using a hardware interrupt generated by a push button (SW0). Each button press toggles the assigned LED.

## Responsibilities

- Configure the button GPIO as an input with edge-triggered interrupt.
- Register and manage a GPIO interrupt callback.
- Toggle a target LED whenever the button interrupt fires.

## Key API

- `void init_button_interrupt_led(const struct gpio_dt_spec *led);`
  - Stores the LED descriptor used by the ISR.
  - Calls the internal `_configure_button_interrupt` helper.

## Behavior

- Uses `DT_ALIAS(sw0)` to locate the button in devicetree.
- `_configure_button_interrupt`:
  - Configures SW0 as `GPIO_INPUT`.
  - Enables `GPIO_INT_EDGE_TO_ACTIVE` interrupt on the SW0 pin.
  - Registers `_button_pressed_isr` as the callback via `gpio_init_callback` / `gpio_add_callback`.
- `_button_pressed_isr`:
  - Toggles the LED assigned in `init_button_interrupt_led`.
